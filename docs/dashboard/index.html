    Dashboard

    Warning

    Do not forget to backup the Superset database. Otherwise, changes to existing visualizations and dashboards are changed or new ones are added

    Setup

    docker-compose up -d superset

    docker-compose exec superset superset db upgrade 

    docker-compose exec superset superset fab create-admin \
    --username admin \
    --firstname Superset \
    --lastname Admin \
    --email admin@superset.com \
    --password admin

    docker-compose exec superset superset init
